# Advent of Code
This repository contains the solutions for the  2025 AoC in Kotlin and Perl.

# How to Run

```bash
cd day_x

# Perl Code (requires Perl and Math::LP module for day 10)
cat input.txt | perl first_star.pl
cat input.txt | perl second_star.pl

# Kotlin Code (requires kotlinc and com.microsoft.z3 package for day 10)
cat input.txt | kotlinc -script first_star.kts
cat input.txt | kotlinc -script second_star.kts
```
